# BobApp
一款可对接SSPanel面板的自定义安卓App，支持v2ray和ssr协议代理切换。app里面内嵌网页面板，免去用户重复登录，可以直接在App里面操作面板。

## 支持的前端
|前端              ||              
|------------------|------------------|
|sspanel-uim（任何主题）	   |√                 |
|v2board            |计划中             |
